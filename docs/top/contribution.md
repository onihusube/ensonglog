# 情報追加・修正、お問い合わせなど

## 推奨する方法

[Github](https://github.com/onihusube/ensonglog)のIssue/Pull Requestからお願いします。ただし、この方法はGithubアカウントが必要になる他、Githubの使い方にある程度慣れが必要です。

情報の追記・修正については直接編集してPRを出して頂けると助かります。その他の問い合わせはIssueを立ててください。

## 匿名フォーム

Githubわからん・アカウント作りたくないなどの場合はこちらのフォームからでもokです。

[ensonglog 問い合わせ箱](https://docs.google.com/forms/d/e/1FAIpQLScDUrsE6ZBYIlzrSYuhxtFXIvex6AGMuJxzHvxQQJcCf8o4Nw/viewform?usp=dialog)

こちらのフォームは特にgoogleアカウント等無くても使用できるはずです。個人識別可能な情報は送信されません。

こちらのフォームにおいては送信後まもなく上記GithubのIssueページに対応するIssueが立ち、そこに内容がコピペされます。管理している人は専らこれを確認します。上記フォームで名前を入力（本名である必要はないです）しておくと、タイトルに名前が入るので識別に若干便利になります。したがって、**内容は誰でも見られる状態になる**のでご承知おきください。
